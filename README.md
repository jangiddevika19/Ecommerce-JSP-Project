#  E-commerce Web Application (JSP + Servlet)

## 📌 Project Description

This is a dynamic E-commerce web application developed using **JSP, Servlets, and JDBC**.
It allows users to register, login, browse products, add items to cart, and manage their shopping.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🛍️ Product Listing
* 🛒 Add to Cart
* ❌ Remove from Cart
* 👤 User Profile
* 💳 Checkout UI

---

## 🛠️ Technologies Used

* Java (JSP & Servlets)
* JDBC
* MySQL (Database)
* HTML, CSS
* Apache Tomcat

---

## 📂 Project Structure

* `src/main/java` → Backend (Servlet, DAO, Model)
* `src/main/webapp` → Frontend (JSP, CSS, Images)

---

## ▶️ How to Run

1. Import project in Eclipse
2. Configure Apache Tomcat
3. Run on server
4. Open in browser: `http://localhost:8080`
